Secondary Storage


Secondary storage is consists of data that is not constantly accessible by the computer system. The devices that go under this category can either be inserted or somehow connected ( for example: plugged in via a USB cord).
Some examples or Secondary storage are:

  • External hard disks
  • Portable flash drives
  • CDs
  • DVDs
The capacity of these devices is measured in megabytes(MB), gigabytes (GB) or terabytes (TB).
