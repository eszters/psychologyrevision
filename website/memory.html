Memory storage in the CPU


There are three main types of memory storage in the CPU, which are all used for different types of data storage. These are:

  • Main memory: The main memory of the computer is also known as RAM, standing for Random Access Memory. It is constructed from integrated circuits and needs to have electrical power in order to maintain its information. When power is lost, the information is lost too! It can be directly accessed by the CPU.
  • Mass storage: Non-volatile, large amounts of data, that are stored on the computer permanently. Thos, obviously, does not include the RAM, which is data that is lost as soon as the computer is shut down.
  • General-purpose registers: These provide the fastest flow of data, however, they are only able to store very little information.


Memory can also be divided into two other categories:

  • Primary storage
  • Secondary storage



Another thing related to the memory of the CPU, and the way data is processed in it, is the stored-program concept. John von Neumann and Alan Turing both proposed the stored program concept in separate publications in 1945. Machine code instructions are fetched from main memory one-at-a-time, decoded and executed in the processor.
