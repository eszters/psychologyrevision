Web Robots


Web robots, also known as web crawlers or spiders are Internet bots, which systematically browses the World Wide Web, typically for the purpose of Web indexing. Such crawlers are used by search engines, such as Google, to make finding pages that are searched for quicker and easier to find.
One problematic process an be parallel web crawling, which is used to scan through multiple web pages at the same time. This uses several, often hundreds of computers at the same time, in order to avoid overload, and to make the work quicker.
